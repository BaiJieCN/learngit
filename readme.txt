
Time : 2017-Sep

Project name : Web Voting

High level Features : 
- As a vote creator, you can create a vote question and possible answers from the website
- As a vote creator, you can modify the vote questions and answers 
- As a vote creator, you can publish the voting
- As a vote creator, you can check the vote result
- As a vote participator, you can response to the vote 
- As a vote participator, you can check the vote result
